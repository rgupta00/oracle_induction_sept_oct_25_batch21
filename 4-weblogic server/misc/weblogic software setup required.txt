Uninstall WebLogic Server 14c (If Installed)
----------------------------------------------
If you've already installed WebLogic using this JAR, follow these steps to uninstall:

Step 1: Locate Oracle Home
Default path is usually like:

C:\Oracle\Middleware\Oracle_Home
Step 2: Run deinstall.cmd

Navigate to:

C:\Oracle\Middleware\Oracle_Home\oui\bin
Run in Command Prompt (Admin):

deinstall.cmd
